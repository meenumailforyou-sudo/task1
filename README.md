Annual Sales Dashboard

An interactive Excel dashboard built using Pivot Tables, Pivot Charts, and Slicers to analyze annual sales performance. The dashboard provides insights into sales trends, customer demographics, top-performing states, and order channels through dynamic visualizations and filters.

Tools Used: Microsoft Excel, Pivot Tables, Pivot Charts, Slicers

Key Features:

Order vs Sales Analysis
Men vs Women Sales Comparison
Top 5 States by Sales
Order Channel Distribution
Age-wise Customer Analysis
Interactive Filters (State, Channel, Category)
